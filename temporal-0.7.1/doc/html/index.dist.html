Welcome to the Temporal PostgreSQL Project Home Page

Tools for maintaining and querying time data in PostgreSQL. It includes temporal data types, functions, operators, opclasses (for indexing), and triggers. Also includes documentation for best practices, examples, and useful queries.

The Temporal PostgreSQLproject is a PostgreSQL Community project that is a part of the pgFoundry.

The pgFoundry page for the project is at http://pgfoundry.org/projects/temporal, where you can find downloads, documentation, bug reports, mailing lists, and a whole lot more.
